Summary:Lattitude vs X

The Purpose of this assignment was to write a python script to mine an API to visualize the weather of 500+ cities across the world of varying distance from the equator. We then used Matplotlib to visualize trends in precipitation,temperature, cloudiness, wind speed in relation to their distance from the equator. This site is the culmination of that data analysis and provides visualizations of the data generated as well as conclusions drawn.
